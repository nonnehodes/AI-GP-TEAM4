How to run:

- Ga naar https://gitlab.com/hu-hbo-ict/ai/v1gp
- scroll naar Requirements en install alle modules 
- Installeer de modules met pip3 en niet pip zodat het voor python3 geinstalleerd word
- Zorg ervoor dat python3 op je pc staat, om te checken, typ python3 in de command line, anders moet je dat nog installeren
- scroll naar Start the Recommendation Service

-open cmd (typ bij je search bar van windows 'cmd')
-met de command cd [directory name] kan je via de command line naar de directories die in de directory staan van waar je je bevind
   (dus als je in C:users\user bevind in de cmd, en de directory "documents" bevind zich in user, kan 
   je met cd documents naar C:users\user\documents (met cd .. kan je terug naar de vorige directory))
- ga daarmee naar de directory waar huw.py staat, dat zou op dezelfde locatie moeten staan als deze README
 (bij mij was het C:\Users\User\Documents\complete_project>) 


-Doe precies wat er staat bij Start the Recommendation Service
- als het niet werkt: wanneer je python -m flask run moet runnen volgend de repo, schijf py -m flask run in plaats daar van
- wanner je Running on http://127.0.0.1:5000/ (Press CTRL+C to quit) ziet, laat deze cmd window open en doe er niks mee.

- open een geheel nieuwe cmd window (laat de vorige open)
- doe alles bij Start the Webshop
- als het niet werkt: wanneer je python -m flask run moet runnen volgend de repo, schijf py -m flask run in plaats daar van
- wanner je Running on http://127.0.0.1:5000/ (Press CTRL+C to quit) ziet, laat deze cmd window open en doe er niks mee.

- typ http://127.0.0.1:5000 in je webbrowser url en open de website
